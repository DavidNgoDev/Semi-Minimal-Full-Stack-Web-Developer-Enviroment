# Essentials Developer Envrioment Setup Script
This is a small powershell script that install essential developer tools on a new vm or production windows 10 build.

## Who is it for?
* Front and Backend Web Developers
* UI/UX Developers
* Kotlin Developers
* Swift Developers
* .NET Developers
* Python Developers
